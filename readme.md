# Walking in

**Author: Bu5dr1v3r**

**Inspired by : Ultra Guo**

说明：

为[0,0,0]和[255,255,255]的纯黑白图像设置一个进入点，并利用渐变色填充非白可视化路径。

Moreover:

可以作为全连接图的填色方案。

目前渐变色方案：plasma

### 0.依赖：

#### python 3环境

matplotlib  PIL  Numpy（理论上没有版本限制）

#### 不会安装？请查看tutor.md
[安装python3与依赖教程]: tutor.md	"tutorial"

### 1.用法：

命令行运行main.py

把要做路径的图像放在根目录下，并根据命令行提示操作。
